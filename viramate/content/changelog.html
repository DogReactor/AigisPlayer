Update Notes

Known Issues

  • Skill cooldowns don't reset properly when using a character with abilities or passives that reset cooldowns
  • The AP and BP meters in the popup are incorrect if you very recently spent AP or BP (to start a quest or join a raid)
  • The popup's item buttons get stuck if a network error occurs
  • Mobile site support is buggy
  • The sidebar does not work if you are using a DMM account
  • Normal attacks don't update the number on the OD gauge
  • The debuff timers flicker briefly when updated
  • The HL raids menu is visible even if your rank is too low to start a HL raid
  • The 'Lite' and 'Standard' resolutions run an older version of the game code. Please either play with 'Animation Settings' set to 'Lite', or with 'Resolution Settings' set to 'High'.
